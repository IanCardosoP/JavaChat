**                 Instrucciones:                  **
**    + `javac *.java` para compilar todo          **
**    + `java Servidor` para iniciar               **
**    + `java. Cliente` para iniciar clientesGUI   **
**    + Asegure iniciar primero Servidor.java      **
**    + Ingrese su usuario cuando se le requiera   **
**    + Inicie tantos clientes como guste!         **
**      (Mas de tres sugeridos para probar PMs)    **
**                                                 **
**          Comandos dentro del JavaChat           **
**   `/clientes` para ver la lista conectados      **
** `@otro_cliente` para enviar PM (PrivateMessage) **
**   `/salir` o cerrar ventana para desconectar    **
